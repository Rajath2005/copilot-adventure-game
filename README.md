# copilot-adventure-game

Simple "Choose Your Own Adventure" CLI game implemented in Node.js.

## Features

- Interactive command-line game using Node's built-in `readline` module
- Multiple choices and branching endings
- Beginner-friendly and easy to extend

## Requirements

- Node.js v12 or newer

## Install & Run

Clone this repository and run:

```bash
node adventure.js
```

## How to play

- Type the number of the option you want and press Enter.
- Use Ctrl+C at any time to exit.

## Extending the game

Add new scene functions to `adventure.js` and connect them using the `ask()` helper.
